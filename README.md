# Análisis de Datos de Física – Atlas Higgs Challenge 2014

Este proyecto realiza un **análisis exploratorio y visualización de datos** del dataset **Atlas Higgs Challenge 2014**, utilizando Python, pandas, matplotlib y seaborn.

## Estructura del proyecto


## Requisitos

- Python 3.10 o superior
- pandas
- numpy
- matplotlib
- seaborn

Todos los paquetes están listados en `requirements.txt`.

## Uso
1. Clonar el repositorio:

```bash
git clone <URL-del-repositorio>

2. Instalar dependencias

pip install -r requirements.txt

3. ejecutar
python src/analysis.py


4. Graficas
python src/visualization.py

5.  Descripción del dataset

- `DER_mass_MMC` → Masa invariante del candidato Higgs  
- `DER_pt_h` → Momento transverse del Higgs  
- `Label` → 1 = señal (Higgs), 0 = fondo (background)
