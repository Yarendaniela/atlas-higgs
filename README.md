# Análisis de Datos de Física – Atlas Higgs Challenge 2014

Este proyecto realiza un análisis exploratorio y visualización de datos del dataset **Atlas Higgs Challenge 2014**, utilizando Python, pandas, matplotlib y seaborn.

## Estructura del proyecto

atlas-higgs/
 data/          
src/           
│ analysis.py
│ visualization.py
README.md
requirements.txt

## Requisitos

- Python 3.10 o superior  
- pandas  
- numpy  
- matplotlib  
- seaborn  

Todos los paquetes están listados en `requirements.txt`.

## Descarga del dataset

El archivo original es muy grande (>100 MB) y **no se sube al repositorio**.  
Descárgalo desde Kaggle: [Atlas Higgs Challenge 2014](https://www.kaggle.com/c/atlas-higgs-challenge-2014/data)

1. Coloca el archivo descargado dentro de la carpeta `data/`.  
2. Asegúrate de que el nombre sea exactamente:  atlas-higgs-challenge-2014-v2.csv

1. Instalar dependencias:

pip install -r requirements.txt

2. Ejecutar analisis
